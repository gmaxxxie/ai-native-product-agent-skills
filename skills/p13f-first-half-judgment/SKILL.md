# 前半段判断 Skill

## 适用场景
- 决定产品方向与范围
- 评估问题价值与AI适用性
- 早期筛选与资源分配

## 输入
| 字段 | 说明 |
|------|------|
| problem_statement | 问题描述 |
| available_solutions | 可能的解决方案列表 |
| ai_capability_assessment | AI能力评估结果 |

## 输出
- 方向判断结论（做哪个/不做哪个）
- 价值排序
- AI适用性决策

## 工作流程
1. **做什么判断**：明确核心要解决的问题边界，识别什么是"不在范围内"
2. **价值判断**：评估问题的紧急性/重要性/市场规模/战略价值
3. **AI适用性判断**：该问题是否适合用AI解决？AI能带来什么增量？

## 注意事项
- 前半段判断决定资源投入方向，错误代价最大
- 要用"淘汰法"而非"加法"——明确什么不做同样重要
