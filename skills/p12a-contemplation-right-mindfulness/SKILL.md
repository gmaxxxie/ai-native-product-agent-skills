# 正念 Skill（Right Mindfulness）

## 适用场景
- 需要建立团队的决策反思机制
- 个人希望提升判断质量
- 重复犯同样的判断错误

## 输入
| 字段 | 说明 |
|------|------|
| recent_decisions | 近期重要决策列表 |
| decision_outcomes | 决策的实际结果 |
| personal_factors | 影响判断的个人因素（情绪/压力/信息等） |

## 输出
- 决策日志模板
- 个人判断模式识别
- 团队反思仪式设计

## 工作流程
1. **决策记录**：建立"决策日志"——记录每个重要决策时的判断、依据、预期
2. **结果复盘**：定期（月/季）对照实际结果，检验判断质量
3. **模式识别**：找出个人/团队反复出现的判断偏差（如过度乐观/锚定效应）
4. **沉淀判断**：将有效的判断经验转化为团队的"判断原则"或"检查清单"
5. **持续练习**：设计简单的日常觉察练习（如"今天最重要的判断是什么？"）

## 注意事项
- 正念不是"冥想"，而是"有意识的觉察"
- 重点是"过程质量"而非"结果正确"——好的判断也可能因运气不好而结果差
