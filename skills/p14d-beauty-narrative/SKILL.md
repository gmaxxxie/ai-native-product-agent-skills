# 故事力 Skill（Narrative Power）

## 适用场景
- 产品介绍/文案/演示缺乏感染力
- 信息堆砌但不成体系
- 想提升内容的叙事美感

## 输入
| 字段 | 说明 |
|------|------|
| content_type | 内容类型（文案/演示/产品/视频） |
| core_message | 核心信息 |
| target_audience | 目标受众 |

## 输出
- 叙事结构设计
- 情感曲线规划
- 信息架构优化建议

## 工作流程
1. **结构选择**：根据内容类型选择叙事结构（问题-解决/英雄之旅/三幕剧）
2. **情绪曲线**：规划内容的情感节奏（铺垫→冲突→高潮→收尾）
3. **信息分层**：设计信息的披露顺序与层次
4. **AI辅助**：用AI生成多个叙事变体，人工选择与优化

## 叙事结构模板
- **问题-解决**：痛点→困境→方案→效果
- **英雄之旅**：召唤→启程→试炼→归来
- **三幕剧**：建置→对抗→解决

## 注意事项
- 故事力不是"编故事"，而是"有节奏地披露信息"
- 好的叙事引导受众的情绪与认知，而非强行说服

## 核心概念

### 概念一：形式感与故事感——美感的两半

绝大多数人学美感都从形式感入手：颜色、构图、字体、留白、比例。但形式感只是美感的一半。另一半是故事感：这个画面背后有没有角色、这个氛围从哪里来、这个场景里发生了什么、这个情绪来自哪个人哪种处境。

形式感决定"看起来对不对"，故事感决定"感觉有没有生命"。一个有形式感但缺故事感的作品，可能被说"构图不错，但感觉空"。一个有故事感但形式感稍弱的作品，往往仍然会被人记住——因为它有一种让人想停下来的吸引力。

很多AI生成的图片形式感都很强——构图对、颜色协调、细节精致——但看起来就是"没有人"，没有情绪，没有时刻，只是一个完美的视觉组合。这就是缺少故事感。

### 概念二：故事力的五个维度

故事力不是指你能写出一个完整的叙事，而是指你能感知并表达以下维度：

**角色感**：这个画面里是谁？不一定需要真实人物，而是一种存在感。即使画面里没有人，你也可以让人感觉到"有人刚刚在这里"或"有人即将来到"。

**处境感**：这个人在什么情境下？"一个人在周一早上的通勤中"和"一个人在放假第一天的早晨"，会产生完全不同的画面氛围。处境不只是背景，它是情绪的来源。

**情绪方向**：不是"高级"或"温暖"这种形容词，而是更具体的——"被理解的轻松""刚刚卸下压力之后的安静""一种不需要表达任何东西也没关系的状态"。情绪越具体，画面越有方向。

**时间感**：清晨的光是清澈的，傍晚的光是温暖的，深夜的光是私密的。同样的场景，不同的时间，传递的感觉完全不同。

**与观众的关系**：观众在什么位置看这张图？会感到被邀请、被观看、还是被提醒了某个自己也有过的时刻？这个关系决定了画面的视角和距离感。

### 概念三：以前按感觉，现在要翻译成故事

在AI之前，有故事感的创作者可以靠直觉工作——脑子里有画面，有感觉，然后一点一点调出来。这个过程不需要把感觉变成语言，因为自己就是执行者。

当AI参与创作，你的感觉仍然存在，但AI不能直接"感受到"你的直觉。你需要把它翻译成某种可以被理解的形式。

**翻译前**："我要一张有情绪的图"
**翻译后**："这是一个人在周日早晨醒来，阳光刚好照进房间，她还没有完全清醒，但已经感到一种难得的放松"

前者是形容词，后者是情境。AI能理解的，是情境。当你能把感觉讲成故事，AI才更可能把故事做成真正成立的画面。

### 概念四：故事力是差异化的一部分

风格可以被复制，prompt可以被抄，但有一样东西很难被复制：你对特定情境的感知方式，以及你把感知转化成表达的方式。

别人可以抄你的画风，但很难抄到你为什么在这个画面里放一个背对镜头的人、为什么这盏灯要偏暗、为什么这个镜头要在沉默里多停一秒。这些判断背后，是你独有的故事感——它是美感里最有个性、也最难被表面化的部分。

### 概念五：故事力的来源——不是天赋，是积累

故事力来自：**深度的观察**（不只看形式，还感受形式背后的情境和人物）、**对情绪的敏感**（能感知一个画面传达的情绪并用语言描述）、**对人的理解**（对某类人的生活状态、内心状态有真实理解）、**自身的经历**（曾经真实感受过某种情绪，所以能准确再现它）。

产品经理的背景在这里很有优势——长期做用户研究的人，通常对"某种用户在某种处境下的感受"有非常具体和准确的想象能力。这种能力，直接就是故事感的来源。

## 分步执行

### 步骤一：叙事结构选择——根据内容类型选框架

根据创作内容类型选择合适的叙事结构：
1. **问题-解决型**：痛点 → 困境 → 方案 → 效果（适合产品介绍、案例展示）
2. **英雄之旅型**：召唤 → 启程 → 试炼 → 归来（适合品牌故事、用户旅程）
3. **三幕剧型**：建置 → 对抗 → 解决（适合视频、长文案）
4. **时刻截取型**：一个瞬间、一个细节、一种存在感（适合视觉创作、海报、社交媒体）

**关键原则**：叙事结构不是套模板，而是选择信息的披露顺序。好的叙事让人在对的时间、以对的方式接收到对的信息。

**输出**：选定的叙事结构和信息披露计划。

### 步骤二：情绪曲线规划——设计情感节奏

规划内容的情感节奏：
1. **起点**：受众在什么状态？（疲惫/好奇/怀疑/放松）
2. **铺垫**：用什么建立连接？（共情/提问/场景代入）
3. **转折**：在哪里出现变化？（新信息/新视角/情感触发）
4. **高潮**：最想让受众感受到什么？（共鸣/顿悟/向往）
5. **收尾**：留下什么余韵？（行动冲动/情感回味/记忆锚点）

**输出**：情绪曲线图和各阶段内容规划。

### 步骤三：故事感输入构建——为AI提供情境框架

把故事感翻译成AI可以理解的输入：
1. **角色**：这个画面里是谁？（不一定需要具体人物，可以是存在感）
2. **处境**：在什么情境下？刚刚经历了什么？
3. **情绪**：想让人感受到什么？越具体越好
4. **时间**：什么时候？清晨/午后/傍晚/深夜？
5. **关系**：这个画面是说给谁的？观众在什么位置？

**对比示例**：
```
没有故事感的输入：
"帮我生成一张有电影感的咖啡图，冷色调，留白充足，高级感。"

有故事感的输入：
"这是一个独自生活的城市人，某个工作日的午后，在自己家里泡了一杯咖啡，
坐在窗边，不做任何事，就是看着窗外。她感到一种罕见的安静，不是因为什么都解决了，
而是因为这一刻什么都不需要解决。安静、私人、有一点孤独但不难过，
更像是一种给自己的礼物。"
```

**输出**：带故事感的AI输入文本。

### 步骤四：AI辅助叙事——生成多个变体，人工选择与优化

用AI生成叙事变体：
1. 用同一个故事框架，让AI生成3-5个不同风格的叙事变体
2. 每个变体侧重不同的情绪方向或叙事节奏
3. 人工选择最接近目标的变体
4. 在选中变体的基础上微调细节

**输出**：3-5个叙事变体和最终选择。

### 步骤五：信息分层与节奏调整——让叙事有呼吸感

对选中的叙事进行精细化：
1. **信息分层**：哪些信息先给，哪些后给，哪些不给
2. **节奏调整**：哪里该快（核心信息），哪里该慢（情绪铺垫）
3. **留白设计**：哪里该给受众消化的空间
4. **细节打磨**：一个关键细节往往比十个普通细节更有力

**输出**：最终版叙事内容。

### 步骤六：复盘与故事力训练——持续积累

每次创作后复盘：
1. 这次的故事感准不准？受众的反馈如何？
2. 哪个细节打动了人？为什么？
3. 哪里可以更有故事感？
4. 记录这个情境和情绪，加入你的"故事感素材库"

**输出**：复盘报告和素材库更新。

## 示例 1

**场景**：一个品牌策划需要为一款助眠产品写社交媒体文案。

**执行过程**：

**步骤1 - 叙事结构选择**：
```
内容类型：社交媒体短文案（Instagram/小红书）
选定结构：时刻截取型——不讲完整故事，只截取一个让人共鸣的瞬间
原因：社交媒体用户停留时间短，需要在1秒内产生连接
```

**步骤2 - 情绪曲线规划**：
```
起点：受众状态——睡前刷手机，有点疲惫，不想看复杂的东西
铺垫（0.5秒）：一个熟悉的场景——深夜的天花板
转折（1秒）：一个细节——不是焦虑，而是一种安静
高潮（1.5秒）：情绪共鸣——"原来不是只有我这样"
收尾（2秒）：余韵——产品自然出现，不突兀
```

**步骤3 - 故事感输入构建**：
```
给AI的输入：

"这是一个普通的周三晚上，11点半。你躺在床上，关了灯，
但还没有睡着。不是因为焦虑，不是因为有什么没做完，
只是安静地躺着，听着窗外偶尔经过的车声。

这个时候，你不需要解决任何问题，不需要回复任何消息，
不需要为明天做任何准备。你只是属于自己的。

我想要传达的感觉是：这种深夜的安静不是孤独，而是一种奢侈。
一天中唯一完全属于自己的时刻。

写一段50字以内的文案，配一张画面描述。"
```

**步骤4 - AI辅助生成与选择**：
```
AI生成5个变体：

#1: "深夜的安静，是给自己最好的礼物。"
#2: "11:30，世界终于安静了。这一刻，只属于你。"
#3: "不是睡不着，是舍不得这一天就这么结束。"
#4: "当所有消息都安静下来，你终于听到了自己的声音。"
#5: "深夜不焦虑，只是想多待一会儿。和自己。"

选择：#3
理由：最有故事感——它不是在描述产品功能，而是在描述一种处境。
"舍不得这一天就这么结束"比"安静"更有画面感，也更准确。
它暗示了白天的忙碌，也暗示了此刻的珍贵。
```

**步骤5 - 信息分层与节奏调整**：
```
最终文案：
"不是睡不着，是舍不得这一天就这么结束。"
配图方向：深夜卧室，微弱的暖光，床头一杯水，窗帘缝隙透进来的路灯光。
产品露出：很小，在床头，不打断画面的安静感。
```

**输出**：最终文案 + 配图方向 + 产品露出策略。

## 示例 2

**场景**：一个设计师用AI生成品牌海报，想让画面更有故事感。

**执行过程**：

**步骤1 - 叙事结构选择**：
```
内容类型：品牌海报（户外+社交媒体）
选定结构：时刻截取型——一个让人想多看几秒的画面
目标：让人感觉到"有人在这里，有故事在发生"
```

**步骤3 - 故事感输入构建**：
```
给AI的输入（对比实验）：

版本A（形式感输入）：
"生成一张咖啡品牌海报，日系极简风格，自然光，留白充足，
高级感，冷色调。"

版本B（故事感输入）：
"这是一个下雨天的下午三点。一个咖啡馆的窗边座位。
桌上放着一杯喝了一半的拿铁，旁边是一本翻开的书，
书页被窗外吹进来的一点风轻轻翻动了一角。

椅子上放着一件外套，说明有人坐在这里，但现在暂时离开了——
可能去续杯了，可能去洗手间了，但很快会回来。

窗外在下雨，但这个角落是干的、暖的、安静的。

我想要传达的感觉：这个咖啡馆是你在下雨天最想待的地方。
不是因为它有多高级，而是因为它让你感到安全。"
```

**步骤4 - AI生成与选择**：
```
版本A结果：5张标准日系咖啡图——干净、精致，但"没有人"
版本B结果：5张有故事感的图——能感觉到"有人在这里生活过"

选择版本B中的一张：
有一张图里，杯子旁的书页刚好被风吹起一角。
就是这个细节，让整张图突然有了生命——
你能感觉到刚才有人坐在这里，现在暂时离开了。

对比：版本A的图你看了就忘，版本B的图你想多看两秒。
那多出来的两秒，就是故事力的价值。
```

**步骤5 - 故事力复盘**：
```
关键发现：
1. "有人刚刚在这里"比"这个画面很好看"更有吸引力
2. 一个不完美的细节（被风吹起的书页）比十个精致的元素更有力
3. 故事感不需要完整叙事，只需要一个可以被感知的时刻
4. 受众的共鸣来自"我也经历过这个时刻"

加入故事感素材库：
- 下雨天+咖啡馆+窗边 = 安全感、属于自己的时间
- 翻开的书+被风吹起的页 = 有人在，但暂时离开
- 喝了一半的咖啡 = 不匆忙，慢慢来
```

**输出**：带故事感的品牌海报 + 故事力复盘 + 素材库更新。
