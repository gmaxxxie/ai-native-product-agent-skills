# AI角色定位 Skill

## 适用场景
- 过度依赖AI生成，失去个人风格
- 不理解AI在审美工作流中的正确位置
- 想保持审美主导权但不知如何与AI协作

## 输入
| 字段 | 说明 |
|------|------|
| ai_usage_pattern | 当前使用AI的方式（直接生成/迭代/辅助） |
| personal_style | 个人风格描述 |
| output_ownership | 对AI输出成果的归属感受 |

## 输出
- AI角色诊断（放大器/陪练器/协作者/替代者？）
- 审美主权保持策略
- 人机协作流程设计

## 工作流程
1. **角色识别**：当前是把AI当工具还是当创作者？
2. **主权检查**：最终判断权在谁手里？标准由谁制定？
3. **协作模式设计**：明确AI的边界与人的控制点
4. **流程固化**：将协作模式固化为工作流

## 三种正确角色
- **放大器**：AI扩展你的能力边界，但方向由你决定
- **陪练器**：AI提供反馈与建议，最终选择权在你
- **协作者**：AI完成部分工作，你负责整合与升华

## 注意事项
- AI不是审美主体——审美判断必须由人做出
- 警惕"AI替你审美"的诱惑，那意味着放弃主导权
