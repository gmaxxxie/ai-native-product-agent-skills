# 组织判断力 Skill

## 适用场景
- 个人判断无法形成组织能力
- 团队决策质量不稳定
- 想建立可持续的判断沉淀机制

## 输入
| 字段 | 说明 |
|------|------|
| team_decisions | 团队重要决策记录 |
| judgment_outcomes | 决策结果与预期对比 |
| team_structure | 团队决策流程与角色 |

## 输出
- 组织判断Checklist
- 决策会议模板
- 判断沉淀机制设计

## 工作流程
1. **判断显性化**：要求团队在决策时明确记录判断依据与置信度
2. **多元化输入**：确保决策集合了不同视角（引入反对者/外部顾问）
3. **决策仪式设计**：设计结构化的决策会议流程（如"预读-讨论-表决-记录"）
4. **复盘机制**：定期对照结果回溯判断质量
5. **原则沉淀**：将有效的判断经验抽象为团队原则

## 注意事项
- 组织判断 ≠ 民主投票——要明确决策责任人
- 判断沉淀是长期过程，需要持续记录与复盘
