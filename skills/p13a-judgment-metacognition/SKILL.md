# 判断力元认知 Skill

## 适用场景
- 不清楚自己当前是否在做"判断"
- 想系统理解判断力的价值与稀缺性
- 需要识别不同类型的判断场景

## 输入
| 字段 | 说明 |
|------|------|
| task_description | 任务/问题描述 |
| available_information | 现有信息量与质量 |

## 输出
- 判断类型识别（策略型/执行型/方向型）
- 判断难度评估
- 判断价值等级

## 工作流程
1. **判断识别**：区分"执行任务"与"需要判断"的场景
2. **难度评估**：评估该判断的不确定性与信息缺口
3. **价值判断**：该判断对产品/业务的影响程度
4. **决策类型归类**：属于九步闭环中的哪一类

## 注意事项
- AI降低了信息获取成本，但放大了判断错误的代价
- 高价值判断往往发生在信息不完整时
