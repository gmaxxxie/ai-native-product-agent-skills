# 美感重塑 Skill（Beauty Redefinition）

## 适用场景
- 面对AI生成内容感到审美疲劳或焦虑
- 不理解"美感"到底是什么，只凭感觉判断
- 需要建立系统性的美感认知框架

## 输入
| 字段 | 说明 |
|------|------|
| ai_generation_experience | 使用AI生成内容的体验描述 |
| aesthetic_anxiety | 是否存在"生成焦虑"（太多选择/失去标准） |
| current_beauty_standard | 当前的美感判断标准 |

## 输出
- 生成焦虑诊断报告
- 六维度美感模型解析
- 美感双轴坐标系定位

## 工作流程
1. **生成焦虑识别**：用户是否因AI生成泛滥而失去判断力？
2. **六维度解析**：将美感拆解为感受力/判断力/选择力/平衡力/故事力/一致性
3. **双轴定位**：判断用户偏好的坐标系（艺术性轴 vs 商业性轴）
4. **标准反思**：当前的标准是否足够清晰？是否可解释？

## 核心概念
- **生成焦虑**：当生成变得容易，判断变得困难
- **六维度美感**：美感是可拆解、可训练的系统能力
- **双轴模型**：艺术美感与商业美感构成两个独立维度

## 注意事项
- 美感不是"好看"——好看只是美学的一个子集
- 判断稀缺是AI时代的美感核心命题
