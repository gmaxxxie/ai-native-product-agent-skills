# 前言与核心命题 Skill

## 适用场景
- 快速理解《AI Beaty》的核心思想
- 向他人解释为什么美感在AI时代更重要
- 建立全书的学习基调

## 输入
| 字段 | 说明 |
|------|------|
| reader_background | 读者背景（设计师/产品经理/开发者） |
| reading_goal | 阅读目的（系统学习/快速入门/问题解决） |

## 输出
- 核心命题精炼
- 学习路线建议
- 关键问题清单

## 工作流程
1. **命题传达**：阐明"生成容易后，美感重新定义"的核心观点
2. **焦虑定位**：帮助读者识别自己的"生成焦虑"程度
3. **价值锚定**：说明美感如何成为AI时代的核心竞争力
4. **路径指引**：指向后续Skill的学习顺序

## 核心命题
- 当AI会生成一切，美感还剩下什么？
- 生成不再稀缺，判断变得昂贵
- 美感不是好看，而是高阶能力系统
- 最终护城河是持续提高的标准

## 注意事项
- 此Skill是入口，不作为独立执行流程
- 重点是让读者建立"美感可训练"的认知
